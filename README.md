# Smart-Digital-Payment-System

## Project Description

The **Smart Digital Payment System** is a C++ based console application that simulates different online payment methods using object-oriented programming concepts. The system allows users to perform transactions through multiple payment modes such as Credit Card, UPI, and Wallet.

This project demonstrates the use of **inheritance** and **polymorphism** by implementing different payment methods under a common structure.

---

## Features

* Multiple payment options (Credit Card, UPI, Wallet)
* Runtime polymorphism using virtual functions
* PIN verification for Credit Card payments
* OTP authentication for UPI transactions
* Balance checking for Wallet payments
* Unique Transaction ID generation
* Loop-based system for multiple transactions
* Transaction counter

---

## Technologies Used

* Programming Language: **C++**
* Concepts: **OOP (Inheritance & Polymorphism)**
* Libraries: `<iostream>`, `<cstdlib>`, `<ctime>`

## How to Run

1. Copy the source code into a C++ compiler (Code::Blocks / VS Code / Dev C++)
2. Compile the program
3. Run the executable file
4. Follow on-screen instructions to perform transactions

---

## Concepts Used

* Inheritance (Base class: Payment, Derived classes: CreditCard, UPI, Wallet)
* Runtime Polymorphism (Function overriding using virtual functions)
* Dynamic memory allocation
* Conditional statements and loops

---

## Output

* User selects payment method
* Enters amount and required credentials (PIN/OTP)
* System processes payment and displays transaction ID
* User can perform multiple transactions
